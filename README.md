## Tips:
### String:
#### Reverse the string in Python:
```python
a = 'abc'
a[::-1] is cba
```
#### How to count a frequency of a character in a string: count()
```python
## Count '1' in a binary string:
'{:032b}'.format(n).count("1")
```

### Number:
#### Converting the int --> 32-bit binary:
```Python
'{:032b}'.format(n)
```
#### Converting the 32-bit binary --> Int:
```Python
int(n,2)
```
